# CRUD with Python and MySQL

A simple CRUD (Create, Read, Update, Delete) application built with Python and MySQL to practice database operations and backend fundamentals.

## About the Project

This project was developed as part of my studies in Software Engineering to understand how applications interact with relational databases.

The system allows users to perform basic CRUD operations, demonstrating how Python can communicate with a MySQL database.

## Technologies Used

- Python
- MySQL
- MySQL Connector (Python library)

## Features

- Create new records in the database
- Read and display stored records
- Update existing records
- Delete records from the database

## Database

The project uses a MySQL relational database with basic table structure for storing and managing records.

## How to Run the Project

1. Clone the repository

git clone https://github.com/nic-pereir/CRUD-Python-MySQL.git

2. Install dependencies

pip install mysql-connector-python

3. Configure the MySQL database connection

4. Run the application

python main.py
